    Replace the branch with master¶

    git checkout draft_branch
git reset --hard master
git push --force

    Other must do a git pull --rebase on the draft_branch after the replace

    Workflow, action, encrypted secrets¶

    1. create access token
      • User -> settings -> developer settings -> personal access tokens -> generate new token
    2. create project/env/org encrypted secret
      • User -> your repositories -> repo -> settings -> secrets -> new repository secret

    3. use the secret in workflow/action

      uses: actions/checkout@v2
with:
    repository: dw-sun/dw-sun.github.io
    token: ${{ secrets.DW_SUN_PAT }}
    path: target

    https://docs.github.com/en/free-pro-team@latest/actions/reference/encrypted-secrets
