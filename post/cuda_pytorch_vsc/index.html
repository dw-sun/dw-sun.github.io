CUDA, PyTorch and VSC Installation on Windows 10
This blog lists the key configuration steps to install CUDA, PyTorch, and VSC
2020-09-01 08:31 folder notes   label AI   GPU   tech   schedule 1 min 25 s.

My old laptop(Thinkpad W520) has a GPU, i want to use it to learn some AI technologies, this notes lists the key steps to configure a Nvidia GPU enabled development environment

1. Review latest online guide -> 2. Find out the GPU Model info -> 3. find out the latest GPU driver version -> 4. determine the CUDA Toolkit version -> 5. Review the selected version CUDA toolkit online guide -> 6. Chose the VSC version and install -> 7. Install the CUDA tool kit -> 8. Chose the the PyTorch version and install

Preparing

Review the CUDA Installation Guide for Microsoft Windows, key points below:

  1. A CUDA-capable GPU
  2. A supported version of Microsoft Visual Studio
  3. A workable NVIDIA CUDA Toolkit version for your GPU

Check GPU model, find driver version, determine CUDA Toolkit, VSC version

  1. Check the GPU model - NVIDIA Quadro 1000M
  2. Find the GPU driver latest version
    • https://www.nvidia.com/Download/index.aspx?lang=en-us
  3. According the driver version number, determine which CUDA Toolkit version is workable for it
    • https://docs.nvidia.com/cuda/cuda-toolkit-release-notes/index.html#major-components
    • My GPU driver highest version is 377.55, so the hightest workable CUDA toolkit version is v8.0.61
  4. According the CUDA Toolkit version, choose the VSC version
    • Change to CUDA Toolkit v8 online documents, review the installation guide
    • https://docs.nvidia.com/cuda/archive/8.0/cuda-installation-guide-microsoft-windows/index.html
    • The CUDA Toolkit version is v8, then i choose Visual Studio Community 2015

Install GPU driver

https://www.nvidia.com/Download/driverResults.aspx/120734/en-us

Install VSC

Install Visual Studio Community 2015. Make sure to include all C++ features

Install CUDA Toolkit

https://developer.nvidia.com/cuda-toolkit-archive https://developer.nvidia.com/cuda-80-ga2-download-archive

Install CUDA toolkit v8.0.61

Install PyTorch(CUDA enabled)

  1. Review get started
    • Find the highest PyTroch version which is available for CUDA toolkit v8. The version is v1.0.0
  2. Download the pip whl packages:
    • https://download.pytorch.org/whl/cu80/torch-1.0.0-cp37-cp37m-win_amd64.whl
  3. Create venv, and install pytorch 
    python -m venv cuda80
source cuda80/scripts/active
pip install torch-1.0.0-cp37-cp37m-win_amd64.whl
pip install torchvision==0.2.1

# this package is needed by pytorch
pip install numpy
  4. Verify the installation 
    import torch

print( torch.__version__ )

x = torch.rand(5, 3)
print(x)

print( torch.cuda.is_available() )
