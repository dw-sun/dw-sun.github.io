Hugo and RobotsTxt

August 27, 2020 - 1 minutes read - 80 words

How Hugo support RobotsTxt

  1. Config config.toml

    enableRobotsTXT = true
[params].env = "production"

  2. run hugo command to generate the web site

    • a robots.txt will be generated in public directory

  3. robots.txt templates are existing in layouts/robots.txt file, or themes/ananke/layouts/robots.txt file

    • the templates would affects the content of the generated public/robots.txt

RobotsTxt samples

  1. Which tells search engines that they are allowed to crawl everything:

    User-agent: *
Disallow:

  2. Which tells search engines that they are NOT allowed to crawl anything:

    User-agent: *
Disallow: /
