How to use Google Analytics

last update: 2020-08-28
created: 2020-08-28

What is Google Analytics

It is used to help you understand how your web site are accessing. With it, you can learn which part of your web is good, or bad; you can know how many user was/are accessing your web, where are they from....; then you can improve the website accordingly.

It is also for Android app, and iOS app.

The steps to use it

  1. login google analytics
  2. Create Account - from Admin menu -> Account -> Create Account
  3. Create Property - from Admin menu -> Property -> Create Property
    • The web site need to be verified by google before it can be analysis

  4. Check

    • access the web site
    • report -> real time

  5. Study the reports to enhance your website
